# Frontend UI User Stories

## Feature: Basic Layout & Navigation
**As a** user
**I want** to see a sidebar and header
**So that** I can navigate the application and see my profile status.

- [x] Sidebar with navigation links (Board, Backlog, Settings)
- [x] Header with project title and user profile placeholder
- [x] Responsive layout structure (Sidebar fixed, content scrollable)

## Feature: Kanban Board View
**As a** user
**I want** to see my tasks organized in columns
**So that** I can understand the status of my work.

- [x] Horizontal scrolling for multiple columns
- [x] Vertical scrolling for tasks within columns
- [x] Task cards displaying Title and Description
- [x] Visual distinction between columns
- [ ] Drag and drop support (Future)
- [ ] Edit task modal (Future)

## Technical Tasks
- [x] Setup React with Vite
- [x] Configure Axios client with interceptors
- [x] Implement component structure (Layout, KanbanBoard)
